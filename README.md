# countries-explorer

Countries Explorer is a React application that displays a list of countries from aroun the world. Users can search for countries by name and filter them by region.

---
## How to run the Application:
1. Clone the repository
   ```bash
   git clone https://github.com/Zahra6217/countries-explorer
2. Navigate to the project folder
   cd the-project-folder
3. install dependencies
   npm install
4. Start the development server:
   npm run dev
5. Open browser and go to the url shown in the terminal (e.g., http://localhost:5173)

### API Endpoints:
Get all countries: https://restcountries.com/v3.1/all
Search countries by name: https://restcountries.com/v3.1/name/{CountryName}
Filter countries by region: https://restcountries.com/v3.1/region/{regionName}
